本项目使用的是Chrome浏览器,下载地址如下：

根据浏览器版本下载对应的驱动，可以通过设置—关于Chrome查看自己浏览器的版本。但在实际选择驱动版本的时候，会发现没有完全对应的版本。例如我的浏览器：Google Chrome 已是最新版本
版本 77.0.3865.90（正式版本），而我选择driver版本：77.0.3865.40  能够成功实现。
